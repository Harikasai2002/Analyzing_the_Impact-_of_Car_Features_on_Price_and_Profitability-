Analyzing the Impact of Car Features on Price and Profitability Using Excel
Project Overview

This project analyzes how various car features influence price, profitability, and market performance using Microsoft Excel. The analysis explores relationships between pricing, engine power, fuel efficiency, body style, and brand positioning to generate actionable insights for automotive market strategy and product optimization.

Objectives
Analyze the relationship between car features and pricing
Study market trends and car model popularity
Evaluate how features impact MSRP and profitability
Identify patterns in fuel efficiency and performance
Perform regression and correlation analysis to understand price drivers
Build interactive visualizations and dashboards in Excel

Tools & Technologies
Microsoft Excel
Pivot Tables & Pivot Charts
Scatter Plots & Trendlines
Histograms & Box Plots
Regression Analysis Tools
Correlation Functions
Data Visualization Techniques

Key Analysis Performed
Market Analysis
Created pivot tables to analyze car model popularity across market categories
Developed combo charts to visualize market trends and distribution

Price & Profitability Analysis
Analyzed price distribution across brands and body styles using histograms and box plots
Compared average MSRP across manufacturers using bar charts
Evaluated profitability trends based on pricing variations

Feature Impact Analysis
Studied impact of features such as transmission type, engine power, and body style on price
Used scatter plots with trendlines to identify relationships between variables
Conducted regression analysis to determine key price drivers

Fuel Efficiency Analysis
Visualized fuel efficiency variations across body styles and model years
Analyzed correlation between MPG, cylinders, and performance metrics
Used heatmaps and line charts for trend identification

Manufacturer Analysis
Built pivot tables to calculate average car prices by manufacturer
Compared brand-wise pricing using bar charts
Analyzed variations in horsepower, MPG, and pricing across brands

Key Insights
Car pricing is strongly influenced by engine power and brand positioning
Certain body styles consistently show higher price ranges
Fuel efficiency varies significantly across model types and years
Horsepower and MPG show a clear inverse relationship in many cases
Premium brands maintain consistently higher average MSRPs
